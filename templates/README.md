# 보유 카드 벌크 업로드 양식 안내

`보유카드_업로드_양식.csv` 를 Excel/구글시트로 열어 한 행에 카드 1종씩 입력 후, 어드민 `/admin/cards/bulk` 에서 업로드합니다.

> 파일은 **UTF-8(BOM) CSV** 입니다. Excel에서 그대로 열고 저장하면 한글이 깨지지 않습니다.
> 구글시트 사용 시: `파일 → 다운로드 → 쉼표로 구분된 값(.csv)` 로 내보내기.

## 컬럼 설명

| 컬럼 | 필수 | 설명 | 입력 예시 |
|---|---|---|---|
| `구단` | ✅ | 아래 "구단 값" 중 하나 | `LG`, `삼성` |
| `선수명` | ✅ | 카드 주인공 선수명 (선수 마스터에 없으면 자동 생성) | `오지환` |
| `종류` | ✅ | 아래 "종류 값" 중 하나 | `노멀(홈)`, `패러렐` |
| `카드번호` | 선택 | 카드 번호/식별자 | `LG-001` |
| `부가구분` | 선택 | 홈/어웨이, 알파벳 글자, 넘버링 등 | `넘버링 25/50` |
| `표시명` | 선택 | 비우면 `선수명 + 종류` 로 자동 표시 | `문동주 루키 오토` |
| `가능수량` | ✅ | 판매/교환 가능 보유 수량 (정수 ≥ 0) | `2` |
| `예약수량` | 선택 | 예약중 수량 (정수 ≥ 0, 비우면 0) | `1` |
| `완료수량` | 선택 | 거래 완료 수량 (정수 ≥ 0, 비우면 0) | `0` |
| `개별가격` | 선택 | 이 카드만의 가격(원). 비우면 종류 기본가 적용 | `8000` |
| `희망여부` | 선택 | `O` = 희망 / 비우거나 `X` = 아님 | `O` |
| `메모` | 선택 | 내부 메모 (공개 상세에도 노출) | `상태 좋음` |
| `이미지URL` | 선택 | 이미지 주소 (Storage 업로드는 추후 개별 편집) | `https://...` |

## 구단 값
`LG`, `KT`, `SSG`, `NC`, `두산`, `KIA`, `롯데`, `삼성`, `한화`, `키움`
(정식명 `LG 트윈스` / slug `lg` 도 허용)

## 종류 값
`노멀(홈)`, `노멀(어웨이)`, `알파벳`, `홀로`, `패러렐`, `승리부적`, `스페셜`, `친필사인`
(코드 `normal_home` `normal_away` `alphabet` `holo` `parallel` `charm` `special` `auto` 도 허용)

## 주의
- 한 행 = 카드 1종. 같은 카드를 여러 상태로 가졌다면 `가능/예약/완료` 수량으로 나눠 한 행에 적습니다.
- 헤더(첫 줄)는 **수정하지 마세요.** 컬럼 순서는 바꿔도 됩니다(헤더명으로 매핑).
- 업로드 시 **미리보기 → 오류 행 확인 → 확정** 순서로 진행됩니다. 오류가 있는 행은 건너뛰고 나머지만 반영할 수 있습니다.
